power + This is the only required input to get an action out of the lamp. It is a binary switch acting as a gateway to the rest of the functions accessible to the user.

intensity + The 1-6 scale allows the user to pick their desired brightness level, visualized here as saturation. Before anything is selected, the lamp will opt to the highest setting by default.
mood + The lamp has five ‘mood’ options to choose from that change the color balance and hue of the lamp. Although using the mood button is not a required input, the lamp will opt for a default if it is not actioned. Unlike intensity, you cannot specifically select a mood; you must cycle through until you reach the desired setting.
output + Using the mood and intensity settings, the lamp banks from a reservoir of 30 possibilities to create the display output.
