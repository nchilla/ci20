power
The power button is the first input and the only required input. It is a gatekeeper that determines what other functions are accessible.
mood
Mood is not a required input, but only because the lamp assigns a default beforehand to determine the starting display. The lamp's interface is such that you are unable to specify the mood you want, and instead are only given the capability to cycle through them.
intensity
Like mood, intensity is optional but assigns a default value. Unlike mood, it has a scale of 6 buttons to choose the desired intensity level.
